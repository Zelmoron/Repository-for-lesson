API Endpoints
--------------

.. http:post:: /upload/{username}

   Загрузка данных пользователя. Для каждого пользователя проверяется, совпадает ли имя и пароль. Если совпадают, данные обновляются.

.. http:get:: /get_information/{username}

   Получение информации о пользователе, включая город, возраст и хобби.

.. http:post:: /registration

   Регистрация нового пользователя. Проверяется, существует ли пользователь с таким же именем.

.. http:get:: /get-users

   Получение списка всех пользователей с их данными.
